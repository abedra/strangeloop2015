Setup

Build a Redis container

            
$ docker run --name repsheet-redis -d redis
$ docker run -it --link repsheet-redis:redis --rm redis \
  sh -c 'exec redis-cli -h "$REDIS_PORT_6379_TCP_ADDR"  \
  -p "$REDIS_PORT_6379_TCP_PORT"'
